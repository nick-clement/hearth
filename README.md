## Deploy to Netlify

### Option 1: Via GitHub (Recommended)

1. Create a new repo on GitHub
2. Push these files to your repo:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/yourusername/hearth-app.git
   git push -u origin main
   ```

3. Go to [Netlify](https://netlify.com)
4. Click "Add new site" â†’ "Import an existing project"
5. Choose GitHub and select your repo
6. Leave all settings as default (no build command needed!)
7. Click "Deploy site"

Your site will be live at `your-site-name.netlify.app` in seconds!

### Option 2: Drag & Drop

1. Go to [Netlify](https://netlify.com)
2. Drag the entire project folder into the deployment area
3. Done! Your site is live
